# Ethers 3D-Concept

## Design

A three dimensional iteration of Ethereum cryptocurrency icon. Made using Autodesk Fusion and OpenSCAD, while keeping in mind the underlying mathematics involved of blockchain technologies.

Rendered views are available in the `/images` directory

![Ethers 3D-Concept Rendering](https://github.com/cipher0x0000/mech/blob/main/_ethers/images/ethers_rendered.png?w=500)
